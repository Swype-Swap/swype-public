# SWYPE Public

This repository is the public-facing documentation and trust surface for SWYPE.

SWYPE is a non-custodial crypto payments and swap-routing platform for creators, communities, freelancers, and internet businesses.

Official links:

- Website: https://swypeswap.com
- X: https://x.com/swypeswap
- Telegram Mini App: https://t.me/swypeswapbot/SwypeSwap
- Support: support@swypeswap.com
- Security: security@swypeswap.com

## What SWYPE Does

SWYPE helps users access crypto-native payment and routing experiences through a cleaner product surface.

Core public positioning:

- non-custodial crypto payments
- public and private swap access
- creator and request-style payment surfaces
- Telegram-native access through the SWYPE Mini App
